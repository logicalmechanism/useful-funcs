addTwoLists :: [Integer] -> [Integer] -> [Integer] #

Add pairs of elements together from two lists into a new list. The length of the list will be the shortest input list. Empty list inputs return empty list outputs.

>>> addTwoList [1,2,3] [3,2,1]
[4,4,4]
